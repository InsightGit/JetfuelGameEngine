Detailed Description

jetfuel::draw::exceptions::Font_not_init_exception

This exception should be thrown when a font passed into a function as an argument is not loaded and ready for the class to use it. This exception is not meant to be used outside of the Text class.

Code Example:

void Do_something_with_font(Font font){ if(font.Is_font_loaded()){ // do something with the font... }else{ throw jetfuel::draw::exceptions:: Font_not_init_exception(); } }
