Detailed Description

An Universal Input System manager to translate SDL Input events into in-game actions. When used in tandem with the Universal Input System (or UIS) interpreter, it allows your players to modify the bindings for certain in-game actions via a simple JSON file, makes implementing a control binding option in-game easier, and allows your program to perform the same action regardless if the player is playing with a Keyboard & Mouse, a controller, or even a touch screen.
