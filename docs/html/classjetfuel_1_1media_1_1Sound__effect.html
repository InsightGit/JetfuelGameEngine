Detailed Description

A sound playback class for short(and sweet) audio files, which supports WAV(.wav), AIFF(.aiff), VOC(.voc), MOD(.mod, .xm, .s3m, and more), MIDI(.mid), OggVorbis(.ogg), MP3(.mp3), and FLAC(.flac).

Code Example:

jetfuel::media::Sound_effect sfx;
if(sfx.load_audio_file("SFX.ogg")){
std::cout << "Unable to load SFX.ogg Error:" << Mix_GetError()
<< std:endl;
return 1;
}
sfx.Play();
