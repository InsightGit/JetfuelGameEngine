virtual Rect2d_int jetfuel::draw::Rectangle_interface::Get_rect_to_draw ( )
pure virtual

Returns the Rect2d to be drawn.

Returns the jetfuel::draw::Rect2d to be drawn when Draw() is called. This is a pure virtual function that must be implemented by the children. This Rect2d_int will be a rectangle of the entire Drawable.
